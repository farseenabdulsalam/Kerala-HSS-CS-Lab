Computer science lab assignments for Kerala higher secondary students

Special files

'makeit' is a bash script I have written to automate compiling and preparing output of the programs. You can safely ignore that file. If you have a linux system, you can modify input, open a terminal in the corresponding folder (eg: cpp) and type ./makeit to create new output

'input' is a file that is automatically input to the compiled program by 'makeit' and the output is stored in respective output folder
